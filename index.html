Que-"Print output: 
var x = 21;
var girl = function () {
    console.log(x);
    var x = 20;
};
girl ();"

ans- The output of the code will be undefined. The reason for this output is due to variable hoisting in JavaScript. When the girl function is invoked, it first logs the value of x. However, within the girl function, there is a local variable declaration var x = 20;.

function girl() {
    console.log(x);
    var x = 20;
};

ans- undefined and 21. the console.log(x) statement inside the girl function logs undefined because the local variable x is hoisted but not assigned a value before it is logged. The console.log(x) statement outside the girl function logs 21 because it refers to the global variable x that was assigned the value of 21.

Write a function that accepts parameter n and returns factorial of n

ans- function factorial(n) {
  if (n === 0 || n === 1) {
    return 1;
  } else {
    return n * factorial(n - 1);
  }
}
